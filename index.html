J

The Programming Language

an overview by Nellie Spektor & Alexander Sosnovsky

Background

  • Created in the 1990s by Kenneth Iverson and Roger Hui
  • Based on APL (from the 1960s) and FP/FL (1970s/1980s)
  • Most often used for mathematical, statistical, and logical analysis of data
  • Front end applications are the console, and the browser
  • Code can be difficult to read. Most characters have two uses: a monadic use and a dyadic use.
    • An example is that “<” is used to put characters in a box, while “<<” is the less than sign
    • In addition, characters such as “{“ and “{{” have entirely different purposes in J

Description

  • J is different from most other languages because it allows and somewhat forces you to think BIG and plan your way through the problem
  • J has labs to help you learn and understand the language (like tutorials).
  • Very good for mathematical and statistical programming
  • Can be used for object oriented programming
  • Free and open source software
  • Array-oriented language

Data Types

  • Numeric
    • Bit: 0
    • A list of bits: 0 1 1 0 1 0 1
  • Literal
    • Literal/Character: 'a'
    • A list of literals / A string: 'abcd'
  • Boxed

    •      Boxes are declared like this: <1 0 0 1 0

    +---------+

     |1 0 0 1 0| 

    +---------+

Lists

You can perform operations on every item in a list

for example, '*:' means square

*: 1 2 3 4

     1 4 9 16

Rank

  • Rank refers to the number of dimensions and is very useful in J
    • A scalar is of rank 0
    • A list of numbers is of rank 1
    • A table of rank 2
    • And so on.
  • >>> M =: 2 3 $ 'abcdef'
  • >>> M
  • abc
    def
  • M has two cells; each of rank 1.

Tacit Programming involves composing functions together and not referring to any variables.

Uses

  • Mathematical, statistical, and logical analysis of data.
  • Function-level and object-oriented programming.
  • Works well for network analysis.
  • Can be used in the case that the user wants to make their code unreadable for anyone who looks at it.
  • Also usable for websites.
  • J also has a relational database management system called Jd.
